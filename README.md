# networth-calculator

## Description

Keep track of your net worth

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ docker-compose up -d

# production mode
npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
